# João Barreto Portfolio

This repository contains a static React/Vite portfolio ready for deployment to AWS S3 + CloudFront.

## Overview

- Site with hero, about, skills, projects and contact sections.
- LinkedIn button available in the header.
- Each project card can link to a GitHub repository.
- GitHub Actions workflow for automatic S3 deployment.
