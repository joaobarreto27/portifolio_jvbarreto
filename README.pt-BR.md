# Portifólio João Barreto

Este repositório contém um portfólio estático em React/Vite preparado para deploy em AWS S3 + CloudFront.

## Descrição

- Site com seções de apresentação, sobre, skills, projetos e contato.
- Link direto para o LinkedIn no cabeçalho.
- Cada projeto pode apontar para o repositório GitHub.
- Workflow GitHub Actions para deploy automático no S3.
